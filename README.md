# SoftwareProject

### How to install

```
npm install
cd client
npm install

to run in localhost

npm run dev

to run tests

npm test
```

### You should create keys.js in config folder.Example config/keys.js:
``` 
module.exports = {
    mongoURI: '...',
    secretOrKey: '...',
    sendgridKey: '...'
}
```
